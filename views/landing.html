Welcome to Climacell's Weather API!

In order to pull forcast data for a specific location, you may use one of the following endpoints:

To get all forcasts for a spcific location with known latitude and longitude values go to: https://climacell-weather-api.herokuapp.com/weather/data?lat=X&lon=Y

To get forcasts metrics for a specific location with known latitude and longitude values go to: https://climacell-weather-api.herokuapp.com/weather/summarize?lat=X&lon=Y

Notice you must replace X and Y in the query string with your lat and lon float values of your location! For example: https://climacell-weather-api.herokuapp.com/weather/data?lat=-180.0&lon=-90.0

Output is presented using json format.
